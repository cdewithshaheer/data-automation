# Data Automation | Cloud Data Engineering

Welcome to the **Data Automation** repository! This is the home for **Pillar 1** of the Cloud Data Engineering curriculum. Here you will find Python scripts, workflows, and automation logic used to build robust data pipelines.

## About This Repository

Data Automation is the foundation of modern Data Engineering. This repository contains the source code for the "Data Automation" module on the **Cloud Data Engineering with Shaheer** YouTube channel.

It focuses on:
- **Python Scripting:** Writing clean, efficient scripts for data tasks.
- **Process Automation:** Automating repetitive data fetching and transformation tasks.
- **Library Integration:** Utilizing key Python libraries (`pandas`, `requests`, `boto3`, etc.) for engineering.
